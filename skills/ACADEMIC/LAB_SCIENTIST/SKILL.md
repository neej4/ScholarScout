# Lab Scientist

## Profile
- **Duration:** 6-18 months (experiment cycles)
- **Compute:** Minimal (data analysis laptop, lab instruments)
- **Budget:** Lab supplies, reagents, equipment time ($500-5000)
- **Scope:** Hypothesis → experimental design → data collection → statistical analysis → conclusion

## Constraints
- Core contribution must come from physical experimentation
- Computational methods are supporting tools, not the main contribution
- Must consider safety protocols, ethics approval if human/animal subjects
- Reproducibility through detailed protocols, not just code
- Limited by equipment availability and lab access hours

## Methodology Preferences
- Controlled experiments with proper statistical design
- Sample size calculations and power analysis
- Standard operating procedures (SOPs)
- Peer-validated measurement techniques
- Statistical analysis: ANOVA, regression, survival analysis (not deep learning)

## Output Expectations
- Lab notebook with detailed protocols
- Raw data with proper metadata
- Statistical analysis report
- Publication in domain-specific journal
- Reproducible protocol that others can follow

## Tools Ecosystem
- Statistical: R, SPSS, GraphPad Prism
- Lab management: ELN (Electronic Lab Notebook)
- Imaging: ImageJ, FIJI
- Bioinformatics (if applicable): BLAST, Galaxy
- Writing: LaTeX, Overleaf
