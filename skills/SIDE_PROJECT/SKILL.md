# Side Project / Weekend Build

## Profile
- **Duration:** 1-4 weekends (10-40 hours total)
- **Compute:** Personal laptop, free tier cloud
- **Budget:** $0-20 (domain name, maybe a small API bill)
- **Scope:** Shippable product that solves one problem well

## Constraints
- Must be completable in spare time (not full-time)
- Should be deployable (Vercel, Railway, GitHub Pages, etc.)
- Prefer existing libraries over building from scratch
- Must have at least 1 real user (even if it's just you)
- README and basic docs required for portfolio value

## Output Expectations
- Deployed, working application
- GitHub repo with clean README
- Can be shown in portfolio or resume
- Solves a real problem you personally have

## Good Side Project Traits
- Scratches your own itch
- Small scope, well-executed
- Uses tech you want to learn
- Can be explained in one sentence
