# Industry R&D Project

## Profile
- **Duration:** 1-6 months (sprint-based)
- **Compute:** Company cloud budget (AWS/GCP/Azure)
- **Budget:** Company-funded
- **Scope:** Prototype that can become production feature

## Constraints
- Must align with company/team OKRs
- IP considerations (can't publish everything)
- Must show ROI to stakeholders (time saved, revenue impact, cost reduction)
- Production-readiness matters (not just accuracy)
- Must work with existing tech stack
- Security and compliance requirements apply

## Output Expectations
- Internal demo to stakeholders
- Technical design doc
- Benchmark results vs current solution
- Migration/integration plan
- Cost analysis (compute, maintenance, opportunity cost)

## What Industry R&D Looks Like
- "Can we replace this manual process with ML?"
- "How do we reduce latency of feature X by 50%?"
- "Build a prototype of [competitor feature] for our platform"
- "Evaluate if [new technology] is viable for our use case"
