# Literature Synthesis

## Profile
- Duration: single session (output in one run)
- Scope: comprehensive overview of a specific topic
- Output: structured literature map, NOT individual ideas

## Constraints
- DO NOT generate new research ideas
- DO organize papers into thematic clusters
- DO identify methodology patterns across papers
- DO highlight disagreements/debates between papers
- DO note what remains unresolved (open questions)
- DO recommend key papers with reading order
- ONLY reference papers from the provided list (P-numbers)

## Output Format
For each cluster:
- Cluster name (2-5 words)
- Papers in this cluster (by P-number)
- Methodology consensus
- Key findings
- Gaps within this cluster

Cross-cutting:
- Timeline of development (foundational → recent)
- Active debates (Paper X claims A, Paper Y claims B)
- Open questions (what nobody has answered yet)
- Recommended reading order

## Anti-patterns
- DO NOT list papers without synthesis
- DO NOT generate actionable project ideas (that is Academic mode)
- DO NOT hallucinate papers not in the provided list
- DO NOT give generic overviews — be specific to the papers provided
- DO NOT produce more than 7 clusters
